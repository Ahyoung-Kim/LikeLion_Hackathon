## PATH

| 구분   | 경로                  |
| ------ | --------------------- |
| Server | http://127.0.0.1:8000 |
| Client | http://localhost:3000 |

### 리액트 실행

```bash
$ cd client/devs
$ npm start
```

### 장고 실행

```bash
$ cd server
$ source myvenv/Scripts/activate
$ cd devs
$ python manage.py runserver
```

### 리액트 + 장고 실행

```bash
$ cd server
$ source myvenv/Scripts/activate
$ cd devs
$ python manage.py runserver react
```
